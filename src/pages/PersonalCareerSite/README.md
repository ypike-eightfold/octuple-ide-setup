## Adding New Pages

1. Create a new folder for your page (e.g., `JobSearch/`)
2. Include the main page component (e.g., `JobSearchPage.tsx`)
3. Add mock data file if needed (e.g., `mockJobData.ts`)
4. Add types file if needed (e.g., `types.ts`)
5. Update `src/App.tsx` with the new route
6. Update `src/pages/Home/HomePage.tsx` to include a link
