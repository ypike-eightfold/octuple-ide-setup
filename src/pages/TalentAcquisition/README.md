# Talent Acquisition

This folder contains example pages for the Talent Acquisition product line.

## Pages

- **CandidateDetails.tsx** - Candidate profile and details view

## Features This Product Covers

- Candidate profiles and details
- Job postings and requisitions
- Application tracking
- Interview scheduling
- Offer management
- Hiring workflows

## Adding New Pages

1. Create a new folder for your page (e.g., `JobPostings/`)
2. Include the main page component (e.g., `JobPostingsPage.tsx`)
3. Add mock data file if needed (e.g., `mockJobData.ts`)
4. Add types file if needed (e.g., `types.ts`)
5. Update `src/App.tsx` with the new route
6. Update `src/pages/Home/HomePage.tsx` to include a link
