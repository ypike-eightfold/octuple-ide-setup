# Talent Management

This folder contains example pages for the Talent Management product line.

## Pages

- **People/** - Employee directory and people search
- **Profile/** - Employee profile pages

## Features This Product Covers

- Employee profiles and directories
- People search and filtering
- Org charts and reporting structures
- Skills and competencies
- Team management
- Performance tracking interfaces

## Adding New Pages

1. Create a new folder for your page (e.g., `OrgChart/`)
2. Include the main page component (e.g., `OrgChartPage.tsx`)
3. Add mock data file if needed (e.g., `mockOrgData.ts`)
4. Add types file if needed (e.g., `types.ts`)
5. Update `src/App.tsx` with the new route
6. Update `src/pages/Home/HomePage.tsx` to include a link
